Program.cs File contains main below 3 functions:

getAntivirusName() => this function will give all the installed anti virus softwares.
CheckDirectory() => this function will check that wazuh-agent directory exists or not.
CheckRegistry() => this function will check that wazuh-agent registry key exists or not.

To check the result just open TempApp.sln file with visual studio and run the project by clicking F5. One console winsow will be opened and that will show the result of all the above functions.
